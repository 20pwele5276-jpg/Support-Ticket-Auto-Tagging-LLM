# Auto Tagging Support Tickets using LLM (Task 5)

## Objective
The goal of this project is to automatically classify support tickets into relevant categories using prompt engineering with a Large Language Model (LLM) approach.

## Approach
Instead of training a full machine learning model, this project uses:
- Prompt engineering techniques
- Rule-based LLM-style classification logic
- Keyword-based interpretation of support tickets

The system reads a support ticket and assigns it to one of the predefined categories.

## Categories
- Technical Issue
- Billing Issue
- Account Issue
- App Bug
- General Query

## Methodology
- Defined sample support tickets
- Created a classification function using logical conditions
- Simulated LLM-based decision making using prompt-style structure
- Generated predicted tags for each ticket

## Tools Used
- Python
- Basic NLP logic
- Prompt engineering concept

## Results
The system successfully classified different support tickets into meaningful categories based on their content.

## Conclusion
This project demonstrates how prompt engineering can be used as a lightweight alternative to traditional machine learning models for text classification tasks.
